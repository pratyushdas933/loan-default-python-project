# loan-default-python-project
This project focuses on building a machine learning model to predict loan repayment behavior using the Lending Club dataset. The goal is to identify whether a borrower is likely to fully repay a loan or default, based on various financial and personal attributes.

Project Overview
Performed data cleaning and preprocessing on raw loan data
Conducted exploratory data analysis (EDA) to uncover key patterns and relationships
Handled missing values, categorical variables, and feature engineering
Visualized trends using Seaborn and Matplotlib
Built a binary classification model to predict loan status (Fully Paid vs Charged Off)
